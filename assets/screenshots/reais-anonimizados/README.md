# Prints anonimizados

Esta pasta reúne imagens demonstrativas para uso público no portfólio.

Os prints foram preparados para preservar a estrutura visual dos sistemas, mas não exibem dados reais da operação. Foram substituídos ou removidos:

- nomes de fazendas, frentes, blocos e arquivos;
- códigos operacionais;
- volumes, totais e percentuais;
- datas de processamento;
- marcas, logos e elementos de identificação da empresa;
- qualquer informação que pudesse expor processos internos.

As telas servem apenas para demonstrar interface, organização das informações, fluxo de uso e padrão dos relatórios.
